
# “小程序敲木鱼”产品需求文档 (PRD)

## 1. 产品概述 (Product Overview)

“小程序敲木鱼”是一款以东方禅意文化为核心，旨在为现代都市人群提供一个便捷、有趣的线上静心、解压工具。通过模拟真实的敲木鱼体验，并辅以丰富的交互和个性化设置，帮助用户在碎片化的时间里获得片刻的宁静与专注。

### 1.1. 目标用户 (Target Audience)

- 追求内心平静、希望缓解焦虑的年轻白领和学生。
- 对传统文化、禅意美学感兴趣的用户。
- 需要短暂休息、集中注意力的创意工作者。
- 寻找新颖、有趣互动体验的泛用户群体。

### 1.2. 核心价值 (Core Value)

- **解压放松**: 模拟敲击木鱼的重复性动作和声音，帮助用户释放压力。
- **专注辅助**: 提供一个无干扰的环境，帮助用户集中注意力，进入心流状态。
- **文化体验**: 以现代科技手段传承和演绎传统文化，提供富有美感的互动体验。
- **趣味互动**: 通过多样化的交互设计，增加产品的趣味性和用户粘性。

## 2. 功能需求 (Functional Requirements)

### 2.1. 核心交互：敲击木鱼 (Core Interaction: Tapping the Wooden Fish)

| 功能点 | 详细描述 | 备注 |
| :--- | :--- | :--- |
| **手动敲击** | 用户通过点击屏幕上的木鱼模型进行敲击。 | 这是最核心、最高频的交互。 |
| **敲击音效** | 每次敲击，木鱼发出清脆、真实的音效。 | 音效应具备高保真度，带来沉浸感。 |
| **敲击动画** | 每次敲击，木鱼模型产生被敲击的视觉反馈，如轻微震动、表面泛起涟漪光效。 | 动画效果需流畅、自然。 |
| **功德/积分记录** | 屏幕上有一个非干扰性的区域（如角落或木鱼下方），实时显示敲击次数或“功德+1”的动画文字。 | 动画效果应简洁、优雅，出现后缓缓消失。 |
| **自动敲击模式** | 提供一个“自动敲击”开关。开启后，系统会以设定的频率自动敲击木鱼，解放用户双手。 | 用户可以自定义自动敲击的间隔时间（如0.5秒/次、1秒/次等）。 |

### 2.2. 进阶交互：手势与体感 (Advanced Interaction: Gestures and Motion)

| 功能点 | 详细描述 | 备注 |
| :--- | :--- | :--- |
| **滑动交互** | 在木鱼模型上左右滑动，木鱼会产生有趣的交互反馈。 | - **左滑**: 木鱼轻微“摇头”，并发出“嗡”的持续音。 <br> - **右滑**: 木鱼轻微“点头”，并发出“咚”的沉稳音。 <br> - **变形效果**: 滑动时，木鱼可产生Q弹的拉伸或挤压变形效果。 |
| **缩放手势** | - **双指捏合（缩小）**: 如果木鱼处于非初始状态（如变形后），缩小手-势使其恢复原样。 <br> - **双指张开（放大）**: 当木鱼被持续放大到一定阈值后，会触发一个“化为虚无”的特效（如碎裂成光点、化为烟雾），然后消失，一段时间后重置。 | 这个交互增加了探索的趣味性，象征“放下”与“回归”。 |
| **晃动手机** | 摇晃手机，木鱼会根据手机晃动的方向和幅度，在屏幕上产生对应的摇摆动画，模拟重力感应。 | 增加产品的“生命感”和物理真实感。 |

### 2.3. 个性化配置 (Personalization)

| 功能点 | 详细描述 | 备注 |
| :--- | :--- | :--- |
| **背景更换** | 用户可以从预设的背景库中选择不同的场景。 | - **静态背景**: 如“幽静禅院”、“山水画卷”、“星空宇宙”等高清图片。 <br> - **动态背景**: 如“细雨涟漪”、“竹林微风”、“袅袅青烟”等循环动画。 |
| **音效选择** | 用户可以更换不同的木鱼敲击音效。 | 提供多种选择，如“清脆”、“沉稳”、“空灵”、“古朴”等。部分特殊音效可通过“功德”解锁。 |
| **木鱼皮肤/模型** | 用户可以更换木鱼的3D模型/皮肤。 | - **基础模型**: 不同材质的木鱼（如檀木、梨花木）。 <br> - **创意模型**: 如“金属赛博风木鱼”、“水晶木鱼”、“小猫/小狗形状的木鱼”等。部分特殊模型可通过“功德”或活动解锁。 |
| **敲击文字自定义** | 用户可以自定义敲击时出现的文字，默认为“功德+1”，可修改为“烦恼-1”、“灵感+1”等。 | 增加个性化和趣味性，满足不同用户的心境需求。 |

### 2.4. 3D渲染与视觉表现 (3D Rendering and Visuals)

| 功能点 | 详细描述 | 备注 |
| :--- | :--- | :--- |
| **3D木鱼模型** | 木鱼形象使用3D模型进行渲染，具有立体感和光影效果。 | 模型应精细，表面有木质纹理，光照下有自然反光。 |
| **光影互动** | 3D模型能对虚拟环境光产生反应。例如，当背景切换为“黄昏”时，木鱼的色调和阴影会相应变化。 | 提升整体视觉的沉浸感和真实感。 |
| **陀螺仪视角** | （可选高级功能）轻微转动手机时，视角会发生微小的偏移，让用户可以从略微不同的角度观察3D木鱼，产生裸眼3D的纵深感。 | 增加产品的精致感和可玩性。 |

### 2.5. 激励与成长体系 (Incentive and Progression System)

| 功能点 | 详细描述 | 备注 |
| :--- | :--- | :--- |
| **功德/积分体系** | 用户的每一次敲击（手动或自动）都会累积“功德”或积分。 | 功德是核心的虚拟货币，用于解锁个性化内容。 |
| **解锁系统** | 累积的功德可用于解锁新的“音效”、“背景”、“木鱼皮肤”等。 | 为用户提供持续使用的动力和目标。 |
| **日历与签到** | 提供一个简单的日历视图，记录用户每天的使用情况（如敲击次数）。连续使用（签到）可获得额外功德奖励。 | 培养用户使用习惯。 |
| **成就系统** | 设立一系列成就，如“初入禅境（累计敲击100次）”、“心如止水（连续7天使用）”、“万物皆空（触发10次放大消失特效）”等。达成后可获得特殊奖励或称号。 | 增加挑战性和荣誉感。 |

## 3. 非功能性需求 (Non-Functional Requirements)

| 类别 | 详细描述 |
| :--- | :--- |
| **性能** | - 应用启动速度快，交互响应流畅无卡顿。 <br> - 3D模型和动画在主流机型上能高效渲染，不导致手机发热严重。 <br> - 资源占用（内存、电量）控制在合理范围。 |
| **兼容性** | 兼容主流的iOS和Android手机操作系统版本。 |
| **易用性** | - 界面简洁直观，核心功能无需引导即可上手。 <br> - 设置项清晰明了，方便用户进行个性化配置。 |
| **稳定性** | 应用运行稳定，在各种常规操作和边缘场景下（如网络切换、来电中断）不易崩溃。 |

## 4. 迭代规划 (Future Iterations)

- **社交分享**: 允许用户分享自己的“功德”日/周报告、或者解锁的稀有木鱼皮肤截图到社交媒体。
- **冥想模式**: 结合背景音乐和引导语，提供一个完整的冥想流程，敲击木鱼作为其中的一个环节。
- **声音组合**: 允许用户将木鱼声与白噪音（如雨声、风声、篝火声）混合播放，创造更丰富的听觉环境。
- **桌面小组件**: 开发手机桌面小组件，无需打开App即可进行简单敲击或查看今日功德。
