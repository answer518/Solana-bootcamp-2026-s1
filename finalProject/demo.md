## 📌 项目简介

这是一个基于Solana的“Phygital”（物理+数字）潮玩IP NFT收藏品项目。我们旨在打破实体潮玩与数字世界的壁垒，为每一个实体潮玩创造一个独一无二的、可验证、可交互的“数字孪生”NFT。

项目通过引入可组合与进化的NFT玩法，解决了普通NFT项目玩法单一的问题，为收藏家提供前所未有的虚实互动体验和收藏乐趣。

## 🛠️ 技术栈

- **区块链:** Solana
- **智能合约:** Rust + Anchor Framework (遵循 Metaplex 标准)
- **前端:** Next.js + TypeScript + Solana Wallet-Adapter
- **去中心化存储:** Arweave (用于图片和元数据)

## 💡 核心功能

1.  **基础铸造 (Mint):** 用户可以连接钱包，铸造一个基础款的潮玩NFT。
2.  **藏品画廊:** 在“我的藏品”页面，用户可以清晰地查看自己拥有的所有NFT。
3.  **组合进化 (核心亮点):** 用户可以选择多个指定的基础款NFT，通过一个特殊的“合成”指令，销毁它们并铸造出一个全新的、更稀有的进化版NFT。
4.  **虚实凭证:** NFT作为未来兑换限量实体玩具或参与线下活动的“数字凭证”。
